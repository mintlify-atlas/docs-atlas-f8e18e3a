---
title: "Quickstart"
description: "Beautify your first JavaScript, CSS, or HTML file in under a minute."
---

<Steps>
  <Step title="Install js-beautify globally">
    Install the npm package globally to get the `js-beautify`, `css-beautify`, and `html-beautify` CLI commands:

    ```bash
    npm install -g js-beautify
    ```

    Verify the installation:

    ```bash
    js-beautify --version
    ```
  </Step>

  <Step title="Beautify a file from the CLI">
    Run `js-beautify` on any JavaScript file. The formatted output is sent to stdout by default:

    ```bash
    js-beautify foo.js
    ```

    To write the formatted output back to the same file:

    ```bash
    js-beautify --replace foo.js
    ```

    To write to a different file:

    ```bash
    js-beautify -o foo-formatted.js foo.js
    ```
  </Step>

  <Step title="Use as a Node.js library">
    Install locally for use as a library:

    ```bash
    npm install js-beautify
    ```

    Then import and use it in your project:

    <CodeGroup>

    ```js CommonJS
    var beautify = require('js-beautify');

    var ugly = 'function foo(){return{a:1,b:2};}';

    console.log(beautify.js(ugly, { indent_size: 2 }));
    // function foo() {
    //   return {
    //     a: 1,
    //     b: 2,
    //   };
    // }
    ```

    ```js ESM
    import beautify from 'js-beautify';

    const ugly = 'function foo(){return{a:1,b:2};}';

    console.log(beautify.js(ugly, { indent_size: 2 }));
    // function foo() {
    //   return {
    //     a: 1,
    //     b: 2,
    //   };
    // }
    ```

    </CodeGroup>
  </Step>

  <Step title="Before and after">
    Here is a before/after example showing what js-beautify produces with `indent_size: 2`:

    **Before:**
    ```js
    function foo(){return{a:1,b:2};}if(condition){doSomething();}else{doOther();}
    ```

    **After:**
    ```js
    function foo() {
      return {
        a: 1,
        b: 2,
      };
    }
    if (condition) {
      doSomething();
    } else {
      doOther();
    }
    ```
  </Step>
</Steps>
