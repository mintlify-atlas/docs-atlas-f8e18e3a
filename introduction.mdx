JS Beautify reformats and re-indents JavaScript, CSS, and HTML source code. It can unpack scripts packed by Dean Edward's popular packer, and partly deobfuscate scripts processed by [javascript-obfuscator](https://github.com/javascript-obfuscator/javascript-obfuscator).

You can try it immediately in your browser at [beautifier.io](https://beautifier.io/), where all options are available through the UI.

## Supported languages

| Language   | Node.js | CLI | Python | Web browser |
| ---------- | ------- | --- | ------ | ----------- |
| JavaScript | Yes     | Yes | Yes    | Yes         |
| CSS        | Yes     | Yes | Yes (via `cssbeautifier`) | Yes |
| HTML       | Yes     | Yes | No     | Yes         |

## Runtimes

**Node.js library** — Install `js-beautify` from npm and call `beautify.js()`, `beautify.css()`, or `beautify.html()` from your application code.

**CLI** — Install `js-beautify` globally from npm to get the `js-beautify`, `css-beautify`, and `html-beautify` executables. Beautified output is written to `stdout` by default.

**Python** — Install `jsbeautifier` from PyPI to beautify JavaScript from Python scripts or the command line. Install `cssbeautifier` to also handle CSS. HTML beautification is not available in the Python version.

**Web browser** — Include the library from a CDN to expose `js_beautify`, `css_beautify`, and `html_beautify` as global functions in your web page.
