## What is JS Beautify?

JS Beautify reformats and re-indents JavaScript, CSS, and HTML. It unpacks scripts packed by Dean Edward's popular packer, and partially deobfuscates scripts processed by `javascript-obfuscator`.

It is available as:

- **npm package** (`js-beautify`) — Node.js library and CLI
- **Python packages** (`jsbeautifier`, `cssbeautifier`) — library and CLI
- **Web library** — embed directly in any HTML page via CDN
- **Online tool** — use it instantly at [beautifier.io](https://beautifier.io/)

## Key features

- Re-indent and reformat minified or ugly code
- Unpack scripts packed by Dean Edward's packer
- Partial deobfuscation of `javascript-obfuscator` output
- Language-specific configuration overrides
- EditorConfig integration
- In-source `/* beautify ignore:start */` and `/* beautify preserve:start */` directives
- Templating language support: Django, ERB, Handlebars, PHP, Smarty, Angular

## Quick example

<CodeGroup>

```bash npm (global CLI)
npm install -g js-beautify
js-beautify foo.js
```

```bash Python
pip install jsbeautifier
js-beautify file.js
```

</CodeGroup>

```js Node.js library
import beautify from 'js-beautify';

const ugly = 'function foo(){return 42;}';

console.log(beautify.js(ugly, { indent_size: 2 }));
// function foo() {
//   return 42;
// }

console.log(beautify.css('a{color:red;}', { indent_size: 2 }));
// a {
//   color: red;
// }
```
