## v1.15.4

## v1.15.2

- Patch SNYK-JS-CROSSSPAWN-8303230 issue brought in through old glob package ([#2328](https://github.com/beautifier/js-beautify/issues/2328))
- Release wheels on PyPI ([#2313](https://github.com/beautifier/js-beautify/issues/2313))
- Fix `ModuleNotFoundError: No module named 'setuptools.command.test'` with latest setuptools release ([#2301](https://github.com/beautifier/js-beautify/issues/2301))
- Python: compatible with setuptools ≥ 72 ([#2300](https://github.com/beautifier/js-beautify/issues/2300))

## v1.15.1

- Turn new Angular templating off by default in HTML ([#2247](https://github.com/beautifier/js-beautify/pull/2247))
- Fix performance regression introduced in v1.15.0 ([#2246](https://github.com/beautifier/js-beautify/issues/2246))

## v1.15.0
